ArchLinux Installation

20-06-2020 — Ecrit par AlrikRr — 3 Minutes

Merci à mattiaslundberg j’ai pu reprendre et modifier sa configuration par défaut.

Téléchargement de l’ISO et Live USB

Premièrement, vous devez télécharger le fichier ISO disponible sur archlinux.org.
Ensuite, vous devea faire une clé USB Bootable (Live USB), il y a une multitude de moyen pour le faire.
Pour ma part, je vais utiliser l’outil dd sous linux.

eidorian@root# dd if=archlinux.iso of=/dev/sdb bs=16M && sync

archlinux.iso est votre fichier ISO est sdb est votre clé USB.
Maintenant, vous avez juste à BOTTER sur votre clé USB. Si ça ne fonctionne pas, désactivez votre Secure Boot.

Installation

Si vous rencontrez un problème, dirigez vous sur Archlinux Wiki.
N’écrivez aucunes commande avant de savoir ce qu’elle fait.
Le guide d'installation officiel contient plus de détails.

Première étape

# Modifier la configuration de votre clavier, passez le en Français AZERTY.
loadkeys fr

# Nous allons assumer que votre Ordinateur est connecté à  internet via WIFI.
wifi-menu

# Verifiez si vous avez bien accès à internet.
ping 8.8.8.8

Partitionnement

cgdisk /dev/sdX

# Partition EFI
1 100MB EFI partition # Hex code ef00

# Partition BOOT
2 250MB Boot partition # Hex code 8300

# Partition Chiffrée
3 100% size partiton # Hex code 8300

mkfs.vfat -F32 /dev/sdX1
mkfs.ext4 /dev/sdX2

Chiffrement

cryptsetup -c aes-xts-plain64 -y --use-random luksFormat /dev/sdX3
cryptsetup luksOpen /dev/sda3 luks

# Création de la partition chiffrée
# Ici nous allons crée une partition ROOT et SWAP
# libre à vous de séparer /home ou d'autres. 
pvcreate /dev/mapper/luks
vgcreate vg_crypt /dev/mapper/luks
lvcreate --size 8G vg_crypt --name lv_swap
lvcreate -l +100%FREE vg_crypt --name lv_root

# Formatage en ext4 et création du SWAP
mkfs.ext4 /dev/mapper/vg_crypt-lv_root
mkswap /dev/mapper/vg_crypt-lv_swap

Montez le système

mount /dev/mapper/vg_crypt-lv_root /mnt
swapon /dev/mapper/vg_crypt-lv_swap
mkdir /mnt/boot
mount /dev/sdX2 /mnt/boot
mkdir /mnt/boot/efi
mount /dev/sdX1 /mnt/boot/efi

Installation

# Pour ne pas être embêter après votre installation, notamment avec les paquets wifi/réseau manquant, pensez à tout inclure ici.
pacstrap /mnt base base-devel grub-efi-x86_64 vim git efibootmgr dialog wpa_supplicant linux lvm2 networkmanager dhcp linux-firmware ranger discord db lib32-glibc  gcc-multilib lib32-gcc-libs

FSTAB

genfstab -pU /mnt >> /mnt/etc/fstab
# Création du /tmp pour ramdisk (ajoutez la ligne suivante dans /mnt/etc/fstab)
tmpfs	/tmp	tmpfs	defaults,noatime,mode=1777	0	0
# Change relatime sur toutes les partitions non-boot en noatime (optimisé pour SSD)

chroot new system and setup

# Enter en chroot sur le nouveau système
arch-chroot /mnt /bin/bash

# Configurez l'horloge
ln -s /usr/share/zoneinfo/Europe/Paris /etc/localtime
hwclock --systohc --utc

# Changez le hostname
echo MYHOSTNAME > /etc/hostname

# Mettez à jour vos locale en Français
echo LANG=fr_FR.UTF-8 >> /etc/locale.conf
echo LANGUAGE=fr_FR >> /etc/locale.conf
echo LC_ALL=C >> /etc/locale.conf

# Ajoutez un mot de passe pour le compte root
passwd

# Ajoutez un utilisateur, retirez l'option -s si vous ne voulez pas utiliser zsh
useradd -m -g users -G wheel -s /bin/zsh USERNAME
passwd USERNAME

# Configurez mkinitcpio avec les bons modules  pour générer notre image
vim /etc/mkinitcpio.conf
# Ajoutez 'ext4' à MODULES
# Ajoutez 'encrypt' et 'lvm2' à HOOKS avant filesystems
# Mettre le clavier en FR pour dévérouiller LUKS
# Ajoutez 'keyboard' et 'keymap' avant encrypt (pensez à retirer le keyboard à la fin) 
mkinitcpio -p linux

Configuration du GRUB

grub-install
# Dans /etc/default/grub modifiez la ligne `GRUB_CMDLINE_LINUX` en `GRUB_CMDLINE_LINUX="cryptdevice=/dev/sdX3:luks:allow-discards" ` et lancez:
grub-mkconfig -o /boot/grub/grub.cfg

Quittez et redémarrez

exit

# Démontez toutes les partitions
umount -R /mnt
swapoff -a

# Reboot et n'oubliez pas de retirer la clé USB !
reboot
