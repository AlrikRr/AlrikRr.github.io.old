ArchLinux Install

2020-06-20 — Written by Eidorian

Thanks to mattiaslundberg, I picked up his installation file and edited a few things.

Download ISO and USB Live

First of all you need to download the Archlinux.iso file from archlinux.org.
Then you can make an USB live, there are a ot of way to do that, for my part I’ll use the linux too dd.

eidorian@root# dd if=archlinux.iso of=/dev/sdb bs=16M && sync

Where archlinux.iso is you ISO file and sdb is your USB device.
Now you just need to BOTT from your USB. If the USB fails to boot, make sure that secure boot is disabled in the BIOS configuration.

Install

If you have any troubles, check the Archlinux Wiki.
Don’t write commands when you don’t know what they do.
The official installation guide contains a more verbose description.

First setup

#Set your keyboard layout / Here french keyboard
loadkeys fr

# This assumes a wifi only system
wifi-menu

# Check internet connection
ping 8.8.8.8

Partitions

cgdisk /dev/sdX

1 100MB EFI partition # Hex code ef00
2 250MB Boot partition # Hex code 8300
3 100% size partiton # (to be encrypted) Hex code 8300

mkfs.vfat -F32 /dev/sdX1
mkfs.ext4 /dev/sdX2

Encryption

cryptsetup -c aes-xts-plain64 -y --use-random luksFormat /dev/sdX3
cryptsetup luksOpen /dev/sda3 luks

# Create encrypted partitions
# This creates one partions for root
# modify if /home or other partitions should be on separate partitions
pvcreate /dev/mapper/luks
vgcreate vg_crypt /dev/mapper/luks
lvcreate --size 8G vg_crypt --name lv_swap
lvcreate -l +100%FREE vg_crypt --name lv_root

# Create filesystems on encrypted partitions
mkfs.ext4 /dev/mapper/vg_crypt-lv_root
mkswap /dev/mapper/vg_crypt-lv_swap

Mount the system

mount /dev/mapper/vg_crypt-lv_root /mnt # /mnt is the installed system
swapon /dev/mapper/vg_crypt-lv_swap # Not needed but a good thing to test
mkdir /mnt/boot
mount /dev/sdX2 /mnt/boot
mkdir /mnt/boot/efi
mount /dev/sdX1 /mnt/boot/efi

Install

# Install the system also includes stuff needed for starting wifi when first booting into the newly installed system
pacstrap /mnt base base-devel grub-efi-x86_64 vim git efibootmgr dialog wpa_supplicant linux lvm2 networkmanager dhcp

FSTAB

genfstab -pU /mnt >> /mnt/etc/fstab
# Make /tmp a ramdisk (add the following line to /mnt/etc/fstab)
tmpfs	/tmp	tmpfs	defaults,noatime,mode=1777	0	0
# Change relatime on all non-boot partitions to noatime (reduces wear if using an SSD)

chroot new system and setup

# Enter the new system
arch-chroot /mnt /bin/bash

# Setup system clock
ln -s /usr/share/zoneinfo/Europe/Paris /etc/localtime
hwclock --systohc --utc

# Set the hostname
echo MYHOSTNAME > /etc/hostname

# Update locale for French
echo LANG=fr_FR.UTF-8 >> /etc/locale.conf
echo LANGUAGE=fr_US >> /etc/locale.conf
echo LC_ALL=C >> /etc/locale.conf

# Set password for root
passwd

# Add real user remove -s flag if you don't whish to use zsh
useradd -m -g users -G wheel -s /bin/zsh MYUSERNAME
passwd MYUSERNAME

# Configure mkinitcpio with modules needed for the initrd image
vim /etc/mkinitcpio.conf
# Add 'ext4' to MODULES
# Add 'encrypt' and 'lvm2' to HOOKS before filesystems
# :w! & :q!  ;)
# Regenerate initrd image
mkinitcpio -p linux

Setup GRUB

# Setup grub
grub-install
In /etc/default/grub edit the line GRUB_CMDLINE_LINUX to GRUB_CMDLINE_LINUX="cryptdevice=/dev/sdX3:luks:allow-discards" then run:
grub-mkconfig -o /boot/grub/grub.cfg

Exit & reboot

# Exit new system and go into the cd shell
exit

# Unmount all partitions
umount -R /mnt
swapoff -a

# Reboot into the new system, don't forget to remove the cd/usb
reboot
