Install Nixos - Beginners

2020-06-19 â€” Written by Eidorian

Thanks to Rayan Lahfa

ISO and VBOX configuration

This tutorial was tested on a virtualbox environment, d’ont forget to enable EFI from the VBOX configuration.

You can download the NIXOS ISO here : nixos.org/download
Don’t forget to check the wiki here : nixos.org

Install

if you have any troubles, google it

For this install I’ll use a 25Gb VirtualDisk.

# Display help - Very important command !  
$ nixos-help  

# root access
$ sudo -i

# Display partitions  
$ lsblk  

# Open parted and free 2GB for SWAP later  
$ parted /dev/sda -- mklabel gpt  
$ parted /dev/sda -- mkpart primary 512MiB -2GiB  
$ parted /dev/sda -- mkpart primary -2GiB 100%  

# Create ESP partition  
$ parted /dev/sda -- mkpart ESP fat32 1MiB 512MiB  
$ parted /dev/sda -- set 3 boot on  

# Cryptsetup
$ cryptsetup luksFormat /dev/sda1
$ cryptsetup luksOpen /dev/sda1 nixos-root

# Check if everything is OK and continue
$ ls -l /dev/mapper

#BTRFS nixos-root
mkfs.btrfs -L nixroot /dev/mapper/nixos-root

# Create SWAP
$ mkswap /dev/sda2
$ swapon /dev/sda2

# Check freespace
$ free -hm

# Create BOOT
$ mkfs.vfat -n BOOT /dev/sda3

# Mount nixos-root on /mnt
$ mount -t btrfs /dev/mapper/nixos-root /mnt

# Init btrfs subvolume
$ btrfs subvolume create /mnt/nixos
$ btrfs subvolume list /mnt

# Umount mnt and mount nix-root with btrfs
$ umount /mnt
$ mount -t btrfs -o subvol=nixos /dev/mapper/nixos-root /mnt
$ cd /mnt
$ ls

# BTRFS create subvolume
$ btrfs subvolume create /mnt/home
$ btrfs subvolume create /mnt/var
$ btrfs subvolume create /mnt/nix
$ btrfs subvolume list /mnt
$ mkdir /mnt/boot

# Mount boot
$ mount /dev/sda3 /mnt/boot
$ ls -l /mnt

# Generate config
$ nixos-generate-config --root /mnt

# Install
$ nixos-install

Les commandes de bases

Here are some basic commands for surviving your first NIXOS installation:

# Rebuild your OS
$ nixos-rebuild switch

# Rebuild your OS and display errors
$ nixos-rebuild switch --show-case

# Search package name
$ nix search <name>

# Install new package
$ nix-env -iA nixos.<name>

All the magic of nixos lies in its configuration and the configuration files you’ll be adding to it! Indeed, configure your Nixos via /etc/nixos/configuration.nix. Once done, you can copy/paste your configuration file to another nixos fraichemetn installer to automatically configure it with your settings!
