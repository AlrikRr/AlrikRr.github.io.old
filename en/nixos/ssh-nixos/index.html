SSH Simple Configuration - Beginners

2020-06-19 — Written by Eidorian

Thanks to Rayan Lahfa

SSH configuration

To edit the SSH config, you need to edit the nixos config file : /etc/nixos/configuration.nix
Don’t forget to read and understand this configuration file !

Enable openssh

First steap, you need to enable OpenSSH Daemon.

[...]
# List services that you want to enable:

# Enable the OpenSSH daemon.
services.openssh.enable = true;
[...]

Enable root login

If you want to enable root login with SSH add those lines.
Be careful, you can’t edit the /etc/ssh/sshd_config file, all the configuration must be done to /etc/nixos/configuration.nix
For more information check : Nixos-permitroot

# Enable root login
services.openssh.permitRootLogin = "yes";

Allow TCP port 22 from firewall

# Open port in the firewall
networking.firewall.allowedTCPPorts = [ 22 ];

Define user and pub key

Last step, you need tu add user and add your public key.

# Define a user account. Don't forget to set a password with ‘passwd’.
users.users.eidorian = {
  isNormalUser = true;
  extraGroups = [ "wheel" ]; # Enable ‘sudo’ for the user.
  openssh.authorizedKeys.keys = ["put your publikey here"];
};
