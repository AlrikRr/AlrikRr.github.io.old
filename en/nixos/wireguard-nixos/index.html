Wireguard configuration - NIXOS

2020-06-19 â€” Written by Eidorian

Thanks to Rayan Lahfa

Wiki link

Check the configuration file here : Nixos-Wireguard

Install and Configure

Install WireGuard with this command.

$ nix-env -iA nixos.wireguard

Once it’s done, you need to creat the Wireguard configuration file.

$ vim /etc/nixos/wireguard.nix

And copy the default configuration file from the wiki link.
You must edit a few things from the configuration file.

Generate keys

You need to generate pub/private key from your server and your android device (client).

# Create server keys
$ umask 077
$ mkdir /etc/wireguard
$ wg genkey > /etc/wireguard/private
$ wg pubkey < /etc/wireguard/private > /etc/wireguard/public

# Create Client keys
$ mkdir -p /tmp/client
$ wg genkey > /tmp/client/private
$ wg pubkey < /tmp/client/private > /tmp/client/public

Don’t forget to edit you wireguard.nix

Edit configuration.nix

You need to add a path to the wireguard.nix file into the imports section from the configuration.nix file.

imports =
  [ # Include the results of the hardware scan.
    ./hardware-configuration.nix
    ./wireguard.nix
  ];

Rebuild nixos and restart service

You must rebuild your nixos configuration file and restart wireguard.

# Rebuild nixos
$ nixos-rebuild switch

# Reload wireguard
$ systemctl status wireguard-wg0.service
$ systemctl restart wireguard-wg0.service

Android peer

If you want to use your wireguard VPN from your Android device, follow theses steps.

First of all, you need to open public port from your home router so that way you can access to you VPN from the public IP.

Create configuration file

This file will be used by your Android device.

$ cat conf_phone

[Interface]
PrivateKey = <put-private-key-client>
Address = 10.220.0.2
DNS = 1.1.1.1

[Peer]
PublicKey = <put-pubkey-server>
Endpoint = <Public IP from your server with the port 50999>
AllowedIPs = 0.0.0.0/0
PersistentKeepAlive = 25

Create QRcode

If you don’t want to write every configuration into your phone, just use a QRcode instead.

# INstall qrencode
$ nix-env -iA nixos.qrencode

# Gen the QRcode
$ qrencode -t ansiutf8 < conf_phone

Now you can Scan this QRcode with the Wireguard application from your Android device.
