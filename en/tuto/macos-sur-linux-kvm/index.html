Install MacOs on Linux with KVM

2020-06-26 â€” Written by Eidorian

How to install MacOs Catalina with KVM on your linux.

Thanks to foxlet for his amazing tutorial : foxlet/macOS-Simple-KVM

Requirements

First of all, you need to install QEMU (3.1 or later), python3 and pip.
MacOs.iso is not necessary for this install.

Feel free to check the github link below for more verbose information.

Install

Get the repo and start the install.

# Run jumpstart.sh script. Default install is Catalina, change the option
# if you want high-sierra or mojave
~ ç«ˆé–€Â» ./jumpstart --catalina

qemu config

Create an empty hard disk. Feel free to change the name and the size.

~ ç«ˆé–€Â» qemu-img create -f qcow2 MyDisk.qcow2 60G

basic.sh edit

Next step is to edit the basic.sh file with your setup.

# Add these line at the end of the file.
-drive id=SystemDisk,if=none,file=MyDisk.qcow2 \
-device ide=hd,bus=sata.4,drive=SystemDisk \

Then run the basic.sh script, if you have any errors, look for typos or syntax errors in the file.

Install MacOs

Select the Boot macOS Install from OS X Base System option.

Wait until MacOs Boot.

Once you see the macOS Utilitiles menu, select Disk Utility

Select the QEMU HARDDISK Media 48.32GB and Erase

Now go back to macOS Utilities and select Reinstall macOS

Continue, select the MacOSX disk and Install

The install can take a while.

The Installation will continue after one reboot !

Once the installation is done, reboot and select Boot macOS from Max OSX

Then you can continue to configure your MacOS.
