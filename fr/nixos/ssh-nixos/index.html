SSH Simple Configuration

2020-06-19 — Ecrit par Eidorian

Merci à Rayan Lahfa

SSH configuration

Pour modifier la configuration de votre SSH,, vous devez modifier votre fichier de configuration nixos : /etc/nixos/configuration.nix
Vous ne pouvez pas modifier le fichier par défault ssh situé dans /etc/ssh/sshd_config.

Activez openssh

Premièrement, vous devez activer le Daemon OpenSSH.

[...]
# List services that you want to enable:
services.openssh.enable = true;
[...]

Activez le login ROOT via SSH

Pour votre première configuration, vous devez vous connecter avec un compte ROOT.
ATTENTION, comme dit précédement, vous ne pouvez pas modifier le fichier /etc/ssh/sshd_config, toute la configuration se fait dans /etc/nixos/configuration.nix
Pour plus d’information : Nixos-permitroot

# Activer Root login
services.openssh.permitRootLogin = "yes";

Autorisez TCP port 22 pour le firewall

# Open port in the firewall
networking.firewall.allowedTCPPorts = [ 22 ];

USER et Clé publique

Créez votre utilisateur. Vous devez ajouter votre clé publique pour pouvoir vous connecter à votre machine Nixos.

# N'oubliez pas de lui attribuer un mot de passe avec ‘passwd’.
users.users.eidorian = {
  isNormalUser = true;
  extraGroups = [ "wheel" ]; # Enable ‘sudo’.
  openssh.authorizedKeys.keys = ["Ajoutez votre clé publique ici"];
};
