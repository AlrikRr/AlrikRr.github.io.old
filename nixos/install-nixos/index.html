Installation de Nixos - Débutant

19-06-2020 — Ecrit par Eidorian — 2 Minutes

Table of Contents

  • ISO et VBOX configuration
  • Installation
  • Les commandes de bases

Merci à Rayan Lahfa

ISO et VBOX configuration

Ce tutoriel a été réalisé dans un environnement VirtualBox, n’oubliez d’activer EFI dans les configuration de votre VM.

Vous pouvez télécharger NIXOS ISO ici : nixos.org/download
N’hésitez pas à jeter un oeil au wiki ici : nixos.org

Installation

Si vous rencontrez des problèmes, Google est votre ami

Pour cette installation, j’utilise un Disk Virtuel de 25GB.

# Afficher l'aide NIXOS - Commande très importante ! 
$ nixos-help  

# Changer pour le clavier fr
$ loadkeys fr

# utilisez le compte root
$ sudo -i

# Affichez les partitions actuelle
$ lsblk  

# Ouvrez l'outils parted et liberez 2GB pour le SWAP  
$ parted /dev/sda -- mklabel gpt  
$ parted /dev/sda -- mkpart primary 512MiB -2GiB  
$ parted /dev/sda -- mkpart primary -2GiB 100%  

# Création de la partition ESP
$ parted /dev/sda -- mkpart ESP fat32 1MiB 512MiB  
$ parted /dev/sda -- set 3 boot on  

# Chiffrement LUKS
$ cryptsetup luksFormat /dev/sda1
$ cryptsetup luksOpen /dev/sda1 nixos-root

# Vérifiez si tout est OK avant de continuer
$ ls -l /dev/mapper

# BTRFS
mkfs.btrfs -L nixroot /dev/mapper/nixos-root

# Creation du SWAP
$ mkswap /dev/sda2
$ swapon /dev/sda2

# Vérification de l'espace libre
$ free -hm

# Création de la partition BOOT
$ mkfs.vfat -n BOOT /dev/sda3

# Montez nixos-root dans /mnt
$ mount -t btrfs /dev/mapper/nixos-root /mnt

# Création des subvolume btrfs
$ btrfs subvolume create /mnt/nixos
$ btrfs subvolume list /mnt

# Démontez /mnt et montez nix-root en btrfs
$ umount /mnt
$ mount -t btrfs -o subvol=nixos /dev/mapper/nixos-root /mnt
$ cd /mnt
$ ls

# BTRFS creation des subvolume
$ btrfs subvolume create /mnt/home
$ btrfs subvolume create /mnt/var
$ btrfs subvolume create /mnt/nix
$ btrfs subvolume list /mnt
$ mkdir /mnt/boot

# Montage du BOOT dans /mnt/boot
$ mount /dev/sda3 /mnt/boot
$ ls -l /mnt

# Générer la configuration
$ nixos-generate-config --root /mnt

# Installation
$ nixos-install

Les commandes de bases

Voici quelques commandes de bases pour survivr à votre première installation NIXOS :

# Rebuild votre OS
$ nixos-rebuild switch

# Rebuild votre OS et afficher les erreurs
$ nixos-rebuild switch --show-case

# Rechercher un paquet à installer
$ nix search <name>

# Installer un nouveau paquet
$ nix-env -iA nixos.<name>

Toute la magie de nixos repose sur sa configuration et les fichiers de configuration que vous allez y ajouter !
En effet, configurez votre Nixos via /etc/nixos/configuration.nix.
Une fois fait, vous pouvez copier/coller votre fichier de configuration sur un autre nixos fraichemetn installe pour automatiquement le configurer avec vos paramètres !
