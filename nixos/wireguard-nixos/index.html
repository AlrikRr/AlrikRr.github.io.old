Wireguard configuration - NIXOS

Lien de WIKI

Le fichier de configuration est disponible ici : Nixos-Wireguard

Installation et configuration

Voici la commande pour installer wireguard.

$ nix-env -iA nixos.wireguard

Maintenant que wireguard est installé, vous devez créer le fichier de configuration.

$ vim /etc/nixos/wireguard.nix

Et copier celui du wiki. Pensez à l’adaptder selon vos besoins.

Génération des clés

Vous devez créer des clés Privéd/Publique pour votre serveur et pour votre Android(client).

# Création des clés serveur
$ umask 077
$ mkdir /etc/wireguard
$ wg genkey > /etc/wireguard/private
$ wg pubkey < /etc/wireguard/private > /etc/wireguard/public

# Création des clés client
$ mkdir -p /tmp/client
$ wg genkey > /tmp/client/private
$ wg pubkey < /tmp/client/private > /tmp/client/public

N’oubliez pas d’ajouter votre clé serveur dans le fichier wireguard.nix

Modification configuration.nix

Vous devez indiquer à Nixos la chemin du fichier wireguard.nix dans la section imports de votre configuration.nix.

imports =
  [ # Include the results of the hardware scan.
    ./hardware-configuration.nix
    ./wireguard.nix
  ];

Rebuild Nixos et redémarrez le service wireguard

# Rebuild nixos
$ nixos-rebuild switch

# Reload wireguard
$ systemctl status wireguard-wg0.service
$ systemctl restart wireguard-wg0.service

Android Connexion

Si vous voulez utiliser votre VPN depuis votre Android, suivez les indications.

Premièrement, si vous voulez accéder à votre VPN depuis l’exterieur, vous devez ouvir un port de votre Box internet à le rediriger sur le port par défault Wireguard.

Création du ficheir de configuration Android

Ce fichier sera utilisé par votre Android pour se connecter à votre VPN.

$ cat conf_phone

[Interface]
PrivateKey = <put-private-key-client>
Address = 10.220.0.2
DNS = 1.1.1.1

[Peer]
PublicKey = <put-pubkey-server>
Endpoint = <Public IP from your server with the port 50999>
AllowedIPs = 0.0.0.0/0
PersistentKeepAlive = 25

Générer un QRcode

Si vous ne voulez pas vous embêter à entrer la configuration à la main, avec l’application wireguard il est possible de scanner un QRCODE. Je vais vous montrer comment créer un QRcode rapidement à partir d’un fichier, ici notre fichier conf_phone.

# Installer qrencode
$ nix-env -iA nixos.qrencode

# Création du QRCODE à partir de conf_phone
$ qrencode -t ansiutf8 < conf_phone

Maintenant, vous pouvez scanner votre QRcode avec l’application XWireguard et vous connecter à votre VPN !
