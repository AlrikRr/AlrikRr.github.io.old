Gitpage with Hugo

2020-06-21 — Written by Eidorian

How to install and configure your first github page with hugo theme.

Install hugo and git

First of all, you need to install hugo and git on your computer.

# Unix
$ sudo apt install hugo git

# MacOs
$ brew install hugo git

Create git repo

You need to create 2 repo git.
The first one will be : <username>.github.io
The second one will be : <username>.blog.github.io
This is very important to use your github username and add github.io at the end, because it will be your web address.

this repo will host your hugo site as an html version.

this repo will host your hugo site configuration.

Setup your hugo env

All these command lines work under linux and macOS, personally I use linux for the examples.

$ mkdir eidorianlinux.blog
# Create your hugo site
$ hugo new site eidorianlinux.blog/
$ cd eidorianlinux.blog

Setup your github env

First, you need to copy the clone link of your blog repo

$ git init

# Paste the clone https here
$ git remote add origin https://github.com/eidorianlinux/eidorianlinux.blog.git
$ git pull origin master

# create .gitignore
$ vim .gitignore
# add public/ on the first line

$ git add .
$ git status


$ git commit -m "Init blog"
$ git push -u origin master

Now you should have something like that on your blog repo :

Chose a theme !

For this exemple I’ll use the same theme as I use on my gitpage right now : hello-friend
Of course you can use the theme you want, the command are the same, check all the theme here : hugo-theme

Install themes

Once you have the theme you want, you have to install it.

$ cd themes/
$ git clone https://github.com/panr/hugo-theme-hello-friend.git

Now the most important part of this install, you need to copy the config.toml from the theme to your config.toml.
For exemple, with hello-friend, there is a config.toml file available on the readme.
The theme doesn’t require any advanced configuration. Just copy config.toml to config.toml file in your Hugo root directory and change params fields.


or :

$ cp themes/hugo-theme-hello-friend/exampleSite/config.toml config.toml

Start hugo server

Now start hugo server and check if everuthing is working !

$ hugo server -w

Deploy your site

Now, we need to deploy the hugo ste into your <username>.github.io repo.

$ cd ../
$ pwd
tuto/

$ git clone https://github.com/eidorianlinux/eidorianlinux.github.io.git
$ cd eidorianlinux.blog
$ hugo -d ../eidorianlinux.github.io/

$ cd ../eidorianlinux.github.io
$ git add --all
$ git commit -m "init web site"
$ git push origin master

Now, your website should be working on your github page at the address : https://<username>.github.io

Deploy script

I made a deploy.sh script to facilitate the deployment of the website.

#!/bin/bash

# Usage : ./deploy.sh

cd <path to your username.blog repo>
echo "Commit message :"
read commit

git add .

git commit -m "$commit"

git push origin master

hugo -d ../<username.github.io>

cd ../<username.github.io>

git add .

git commit -m "$commit"

git push origin master

That’s it, it’s all done, I hope you enjoyed this little tutorial!
